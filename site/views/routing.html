Routing

URL's are routed to controller actions automatically. The default route is {controller}/{action}, but can be configured as needed. 

        Go.config({

            //Routes
            routes: [
                //Home route
                {
                    route: "/", //Route definition
                    defaults: { controller: "app", action: "start" }, //Default values
                    require: "controllers/app" //Ensure controller is loaded
                },
                //Another route
                {
                    route: "/myroute/{action}/{anotherparam}", //Route definition
                    defaults: { controller: "mycontroller" }, //Default values
                    require: "controllers/mycontroller" //Ensure controller is loaded
                }                
                //Default route
                {
                    route: "{controller}/{action}"
                },
            ]

        });
