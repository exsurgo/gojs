The basic functionality centers around controllers, actions and routes. GoJS automatically activates links and forms, and routes them to controller actions. The default route is {controller}/{action}.

When a user clicks the below link, the click event is intercepted by GoJS and routed to the controller action. So the URL /message/speak is matched to the default route /{controller}/{action}, which resolves to the MessageController.speak action. The "speak" action has two properties, "remote" which is the remote URL to request, and "response" which is an event to handle the response.

Controller & Action 
        var MessageController = new Controller("message", {
        
            speak: new Action({
                remote: "/api/message/get",     //Connect to remote service
                response: function(e) {         //Handle the response
                    alert(e.data.message);
                }
            })
