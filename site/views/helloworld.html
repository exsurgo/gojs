The following is a simple example of how one would display a simple message returned from the server.

When a user clicks the below link, the click event is intercepted and routed to the controller action. The URL /message/speak is matched to the default route /{controller}/{action}, which resolves to the MessageController.speak action. The "speak" action has two properties, "remote" which is the remote URL to request, and "response" which is an event handler for the response.

Server Response 
        {"message": "Hello World!"}
Controller & Action 
        var MessageController = new Controller("message", {
        
            speak: new Action({
                remote: "/api/message/get",     //Connect to remote service
                response: function(e) {         //Handle the response
                    alert(e.data.message);
                }
            })

        });
