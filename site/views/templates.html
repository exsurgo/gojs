Templates

GoJS includes a full template rendering engine. The following syntaxes are supported.

Inserting Values

Insert Value 
        {title}

Conditionals

If/Else via Tags 
        {? isTrue}
            Its true
        {:}
            Its false
        {/?}
If/Else via Attributes 
        Its true
        Its false

Looping

Each via Tags 
        {~ items}
            {title}
        {/~}
Each via Attributes 
        {title}

Child Templates

Render the child template "child-template" 
        

            {> "child-template", this}
