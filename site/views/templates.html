Templates

GoJS includes a full template rendering engine. The following syntaxes are supported.

Inserting Values

Infer value from current context 
        {title}
Same as above 
        {= this.title}
Dot notation can be used 
        {title.toLowerCase()}

Conditionals

If/Else via Tags 
        {? isTrue}
            Its true
        {:}
            Its false
        {/?}
If/Else via Attributes 
        Its true
        Its false

Looping

Each via Tags 
        {~ items}
            {title}
        {/~}
Each via Attributes 
        {title}

Child Templates

Render the child template "child-template" 
        

            {> "child-template", this}
