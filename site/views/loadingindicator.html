Loading Indicator

GoJS also includes a loading indicator that is displayed when an HTTP request is made or when resources are being loaded.

Show Loading Indicator | Hide Loading Indicator

The CSS style for the loading indicator must be set by the config value loadingIndicatorCSS. 

        .loading-indicator 
        { 
            width: 55px; 
            height: 55px; 
            opacity: 0.4; 
            filter: alpha(opacity=40); 
            text-align: center; 
            z-index: 9999999; 
            background: url(/gojs/site/images/loading.gif) no-repeat center center; 
        }​
