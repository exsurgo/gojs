Links

All relative links are automatically hijaxed. When clicked, a request is made, and a partial HTML update is returned.

Note that all links can be hijaxed, or only selective ones via data-ajax="true". Also note that links are automatically converted to "ajax links" with hash URLs.

External Links

External links are not hijaxed. They also open in a new window. (target="_blank")

Form Posts

Forms are also automatically hijaxed. Forms are also temporarily disabled to prevent double submits. The form submit below would automatically serialize the form, and route the values to the task controller and save action. Submits are local only unless a "remote" property is specified on the controller.
