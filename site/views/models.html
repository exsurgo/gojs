Models

The model property on an action indicates what model should be used. 

        var MessageController = new Controller("message", {

            speak: new Action({
                model: MessageList, //Model to use as context for view
                view: "list"
            })

        });

Response Data

Models can be plain objects or functions. If an object, any response data is merged into a single object. If a function, it is executed with the ActionEvent "e" as the argument. In this case, you can access the response data via the e.data property. 

        var MessageList = function(data) {
            var model = {};                 //Create model to return
            model.messages = e.data;        //Add response data to model
            model.count = e.data.length;    //Add additional properties
            return model;                   
        };
