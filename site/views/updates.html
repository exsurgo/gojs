Updates

GoJS automatically requests content and data from the server, and updates the HTML in the DOM. You can configure an updater in action like so. 

        var ExampleController = new Controller("example", {

            updateContent: new Action({
                updater: "content",
                title: "Content Update",
                view: "content"
            })

        });

Built-in Updaters

There are several built-in updaters. Any of the following can be be used out-of-the-box.

  • content - Adds content in the main content area.
  • insert - Adds content into a specified target element.
  • append - Adds content to the end of an element.
  • prepend - Adds content to the beginning of an element.
  • before - Adds content before a specified element.
  • after - Adds content after a specified element

Custom Updaters

You can also add a custom updater using the Go.addUpdaters() method. The following updater example displays content in a model window using the jQuery UI Dialog plugin.

Add Updater 
        Go.addUpdaters({

            window: function (e) {
		
		        //Require jQuery UI
                Go.require({ scripts: "jqueryui", styles: "jqueryui" }, function () {

                    //Show window
                    e.element.dialog({
                        title: (e.action && e.action.title) || "",
                        modal: true
                    });

                });
		
            }

        });
Use Updater 
        var ExampleController = new Controller("example", {

            showWindow: new Action({
                updater: "window", 
                title: "A Window Update Example",
                view: "sampleview"
            })

        });
